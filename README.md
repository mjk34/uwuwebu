# uwuwebu

Companion website for **professor-rs** — a Discord bot economy system. Shows user profile data, leaderboards, and an interactive 3D ASCII UwU globe.

**Live site:** [https://mjk34.github.io/uwuwebu/](https://mjk34.github.io/uwuwebu/)

## Stack

- Next.js 16 (App Router, static export)
- TypeScript, Tailwind 4
- Deployed via GitHub Pages

## Dev

```bash
npm run dev     # dev server on :3000
npm run build   # production build
npm run lint    # ESLint
```
